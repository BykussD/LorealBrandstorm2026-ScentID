Scent ID: L'Oréal Brandstorm Project
"Scent ID transforms memories and traits into a tangible scent identity. Moving beyond traditional methods, it offers an innovative way to find and identify with your scent. We’re here to revolutionize the fragrance industry through this vision."

Overview
Welcome to the official repository for Scent ID, developed for the L'Oréal Brandstorm challenge. This project bridges the gap between Data Science and Beauty Tech by using Natural Language Processing (NLP) and Machine Learning to translate human emotions, memories, and personality traits into personalized fragrance recommendations.

The Problem
Consumers often face "decision paralysis" when choosing a perfume due to the overwhelming number of options and the abstract nature of scents. Traditional methods fail to build a deep, psychological connection between the user and the fragrance.

Our Solution
We designed a system that replaces traditional scent testing with an AI-driven belonging model. By analyzing a user's psychological profile, Scent ID provides:
A unique, tangible "Scent Identity".
Highly accurate L'Oréal perfume recommendations.
A sophisticated social tool for users to discover and share their personality traits through scent.

Methodology & Tech Stack
Psychological Analysis & Data Collection: Conducted a comprehensive survey with ~100 regular perfume users to map the correlation between personal memories, emotional triggers, and fragrance families.
Machine Learning & NLP: Developed a custom model that processes subjective text inputs (memories, feelings) and categorizes them into objective data points to find the perfect scent match.

MVP Validation & Key Results
High Accuracy: Achieved a 75% accuracy rate in predicting the correct fragrance match, even with a limited initial dataset.
Consumer Behavior: Successfully reduced decision-making time while increasing user engagement and the sense of belonging.

Scalability: Built as a digital-first, low-marginal-cost solution accessible to anyone with a smartphone.
